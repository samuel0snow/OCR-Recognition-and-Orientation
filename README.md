# OCR-Recognition-and-Orientation
用户输入图片和关键词，程序通过OCR技术识别图片中是否包含关键词并且定位关键词出现的位置。
Users input an image and keywords, and the program uses OCR technology to detect the presence of the keywords in the image and locate their positions.

本项目使用了百度开源的PaddleOCR-VL，做到能够离线部署到本地使用。
This project adopts Baidu's open-source PaddleOCR-VL, enabling offline deployment and local usage.
